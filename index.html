Noughts & Crosses game

This game was created to help me practice JavaScript and manipulate the DOM while honing my vanilla JavaScript skills in rendering dynamic page interactions without altering the browser by activating and deactivating CSS codes. Overall, the project allowed me to develop my skills while creating an engaging and interactive game.

Numbers in spanish

"The Numbers in Spanish" was created as a helpful tool for my daughter to learn and practice the numbers 1 to 10 in Spanish. Built using JavaScript and jQuery, this web application features interactive animations and audio playback for each number to engage and reinforce learning. With keyboard support, the application is designed to be simple and user-friendly. In addition to helping my daughter learn Spanish, the project was created with the goal of practicing and understanding basic concepts of JavaScript and jQuery, including event listeners, DOM interaction, and writing DRY code using functions.

Inventory

The Inventory Web Application is a Python and Django-based web tool for managing inventory. Users can create, edit, and delete suppliers, products, purchase orders, clients, and client orders. I initially developed a desktop app with Python and Tkinter but faced difficulties converting it to an executable file. I recreated it as a web app to deepen my understanding of the Django framework and found it easier to work with compared to Flask. I'm happy with the project's outcome and what I learned.

QuizApp

The Quiz App is a JavaScript-based web app created for a friend who enjoys quizzes. It fetches trivia questions and answers from an external API and checks user-selected answers for correctness. It also features event listeners for buttons, audio feedback for correct and incorrect answers, and score management functions. The project was developed to improve JavaScript skills and learn about using external APIs, while also practicing app functionality and understanding the app workflow and lifecycle.
